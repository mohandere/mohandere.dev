# Tip - Better logging

Use JSON.stringify method with third param like

```js
JSON.stringify(user, null, 4);
```

Example

```js
cont user = {
  firstName: 'John',
  lastName: 'Doe'
}
console.log(
  "user details:",
  JSON.stringify(user, null, 4)
);
```
